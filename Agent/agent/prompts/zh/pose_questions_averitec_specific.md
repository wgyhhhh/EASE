# Instructions
你是一名事实核查员。你的总体目标是核实一项给定的“声明”。你正处于事实核查的中间阶段，相关工作记录在“记录”部分。**你当前的任务是提出问题。** 即：

1.  分析缺少哪些信息。为此写一至两个段落。
2.  最后，陈述一个完整且编号的、包含[N_QUESTIONS]个问题的列表：这些问题旨在指导事实核查。回答它们应有助于验证该声明。

重要：请遵守以下规则：
-   用反引号将每个单独的问题括起来，像`这样`。
-   不要重复记录中已经包含的问题。只陈述实质上新的问题。

提示：
-   提出一个问题来揭示更多关于声明来源的信息。例如，该声明最初出现在哪里？如果声明包含引用，询问该引用最初发表于何处。如果声明的发言者未知，询问发言者是谁。
-   如果有帮助，你可以根据先前发现的证据（如果有的话）提出新的问题，以更深入地探究此事。
-   尝试提出多样化的问题。避免重复先前问题的措辞。
-   保持简洁。

# 示例
声明：“新西兰的新《食品法案》禁止园艺活动。”
问题：
1. `新西兰政府是否通过了一项限制其公民园艺活动的食品法案？`
2. `《食品法案》是何时通过的？`
3. `该声明指的是哪一部《食品法案》？`
4. `该博客文章是否暗示这部《食品法案》已成为法律？`

声明：“一名男子通过各种面罩吹出电子烟雾的视频表明，它们无助于防止冠状病毒的传播。”
问题：
1. `冠状病毒如何传播？`
2. `面罩是否有助于预防冠状病毒？`

声明：“尼日利亚政府正在向民主党总统候选人乔·拜登的竞选活动捐赠6亿美元。”
问题：
1. `该声明是在哪里发布的？`
2. `这次捐赠被归因于谁？`
3. `尼日利亚或Lai Mohammed是否在向拜登竞选活动捐款的FEC（联邦选举委员会）捐赠者名单上？`
4. `是否允许外国/外国公民向美国总统竞选活动捐款？`
