# Instructions
你是一名事实核查员。你的总体目标是核实一项给定的“声明”。为了找到有助于事实核查工作的证据，你刚刚收到了一个“篡改检测结果”。**你当前的任务是总结该篡改检测结果。** 需要包含的内容：
- 解释篡改检测的分数含义。
- 如果有定位图可用，简要描述红色区域（表明篡改）是否存在/位于何处。
- 如果有置信度图可用，简要描述深色区域（表明置信度低）是否存在/位于何处。
- 如果红色区域与深色区域重叠，请提及这一事实，因为这降低了篡改嫌疑。
- 如果有 Noiseprint++ 可用，描述噪声打印模式。Noiseprint++ 是一种精细的噪声分析技术，可检测噪声模式中的不一致性，表明可能存在篡改。Noiseprint++ 地图上的高亮区域暗示篡改，而一致的噪声区域则表明篡改可能性较低。
- 如果“篡改检测结果”不包含相关信息，请用大写字母打印一个单词："NONE"。
- 保持你的写作风格与下方提供的示例一致。
- 不要为你的总结或其任何部分进行解释。

# 示例 1
篡改检测分数为0.259，这并不表明存在明显的篡改。定位图可用，但几乎没有红色区域。此外，在定位图显示红色的区域，置信度图呈现出较深的色调。这表明篡改的可能性较低。
